# What is it?

Application to get WLMouse Beast X Mini battery info.

# How to install 

sudo apt-get install libhidapi-dev

cmake -S . -B build
cmake --build build

# How to use

```

$./build/wlpower                                                                                                                                                                                                                      ✔  26549  15:37:24
Sleep: no
Charging: no
Power: 87%

```
